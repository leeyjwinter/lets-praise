# Let's Praise!

## Project Overview
"Let's Praise!" is a web application that allows users to input a person's name and a compliment. Each input generates a floating balloon on the web page. When a balloon is clicked, it displays the person's name and the compliment.

## Project Structure
```
lets-praise

│   ├── css

│   └── js

└── README.md             # Project documentation
```

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, etc.)
- Basic understanding of HTML, CSS, and JavaScript

### Installation
1. Clone the repository or download the project files.
2. Open the `src/index.html` file in your web browser.

### Usage
1. Enter a person's name in the input field.
2. Enter a compliment in the second input field.
3. Click the "Add Balloon" button to generate a floating balloon.
4. Click on any balloon to see the name and compliment displayed.
